Que es una pagina web?

Páginas web, páginas electrónicas, páginas digitales o páginas de red es un documento electrónico o información que puede contener texto, sonido, video, programas, enlaces, imágenes, hipervínculos y muchas otras cosas, adecuado para la llamada Web "mundial". y se puede acceder a través de un navegador web. Esta información suele estar en formato HTML o XHTML y puede proporcionar acceso a otras páginas web a través de enlaces de hipertexto. También suelen incluir otros recursos como hojas de estilo en cascada, scripts, imágenes digitales, etc.
